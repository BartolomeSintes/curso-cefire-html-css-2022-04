margin y padding en html y body

En la hoja de estilo de esta página, las etiquetas html y body tienen definida la propiedad background-color:

html {
  background: url("cuadricula-transp.gif") lightblue;
  margin: 10px;
  padding: 10px;
}

body {
  background: url("cuadricula-transp.gif") lightyellow;
  font-family: sans-serif;
  font-weight: bold;
  margin: 10px;
  padding: 10px;}

h1 {
  border: black 2px solid;
  margin: 0;
}
