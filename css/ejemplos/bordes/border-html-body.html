border en body y html

En la hoja de estilo de esta página, las etiquetas html y body tienen definida las propiedades border, margin y padding:

  • La etiqueta html tiene borde de color rojo y el color de fondo de color rosa.
  • La etiqueta body tiene borde de color azul y el color de fondo de color azul claro.
  • Tanto html como body tienen los mismos márgenes exteriores e interiores (50 píxeles).
