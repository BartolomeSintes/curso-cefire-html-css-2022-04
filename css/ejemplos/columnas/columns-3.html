Aunque la recomendación CSS 3: Columnas no ha sido aprobada, los navegadores admiten la mayoría de propiedades de esta recomendación (con el nombre oficial o prefijadas). Las columnas suponen un gran avance para el diseño de páginas web ya que facilitan la lectura de textos largos en una misma página, sobre todo en las pantallas de escritorio actuales, de cada vez más resolución horizontal.
