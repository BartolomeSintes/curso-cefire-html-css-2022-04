Como muestra el ejemplo siguiente, si la página es una página vacía, los navegadores no interpretan el valor round. Pero si la página tiene contenido, como en este ejemplo, o si el elemento tiene un tamaño definido, el valor round deforma la imagen de fondo de manera que se vea un número entero de veces en cada dimensión (en vertical, con respecto al contenido, no respecto a la página completa).
