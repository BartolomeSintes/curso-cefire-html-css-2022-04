Estos ejercicios tratan sobre los temas tratados en la Semana 2 del curso.

Para facilitar la realización de los ejercicios, se proporcionan unas plantillas de los ejercicios Primeras páginas. Una vez descargado el fichero zip:

  1. Descomprima las plantillas en la carpeta ejercicios. Se creará una carpeta primeras-paginas-plantillas.
  2. Renombre la carpeta primeras-paginas-plantillas como primeras-paginas.

Las capturas de pantalla están tomadas con un ancho de ventana de 1024px aproximadamente.

Primeras páginas 2 - Corregir errores en página web y hoja de estilo

  1. Abra en Visual Studio Code la página pp-2.html.
  2. Corrija los errores sintácticos de la página web y los indicados por VSCode.
  3. Formatee la página web con el atajo Alt+Mayús+f (en Windows) o Ctrl+Mayús+i (en LLiureX).
  4. Abra en Visual Studio Code la hoja de estilo pp-2.css.
  5. Corrija los errores sintácticos de la hoja de estilo y los indicados por VSCode.
  6. Formatee la hoja de estilo con CSScomb con el atajo Alt+Máy+c Alt+Máy+f (en Windows) o Ctrl+Mayús+c Ctrl+Mayús+i (en LliureX).
  7. Abra la página pp-2.html en un navegador y compruebe que el contenido se ve como en la captura siguiente:
